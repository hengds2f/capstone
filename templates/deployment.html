Our application exposes a RESTful API at /api/*. Here are the available endpoints:

MethodEndpointDescription

POST/api/model/trainTrain ML model

GET/api/validateRun data validations

API Usage Example

JavaScript
// GET request with query parameters
const response = await fetch('/api/housing?town=BISHAN&flat_type=4%20ROOM');
const data = await response.json();
console.log(data.data);  // Array of HDB records

// POST request with JSON body
const trainResponse = await fetch('/api/model/train', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({model_type: 'random_forest'})
});
const result = await trainResponse.json();
console.log(`R² = ${result.metrics.r2_score}`);
Python
import requests

# Using the API from Python
base_url = "http://localhost:5000"

# Get housing data
resp = requests.get(f"{base_url}/api/housing", params={"town": "BISHAN"})
data = resp.json()["data"]

# Train a model
resp = requests.post(f"{base_url}/api/model/train",
                     json={"model_type": "linear_regression"})
metrics = resp.json()["metrics"]
print(f"Model R²: {metrics['r2_score']:.4f}")

📖 Flask Best Practices

Python
# 1. Application Factory Pattern
def create_app():
    app = Flask(__name__)
    app.config.from_object('config')
    # Register blueprints
    from api.routes import api_bp
    app.register_blueprint(api_bp)
    return app

# 2. Error Handling
@app.errorhandler(404)
def not_found(e):
    return jsonify(error="Resource not found"), 404

@app.errorhandler(500)
def server_error(e):
    return jsonify(error="Internal server error"), 500

# 3. Input Validation
from werkzeug.utils import secure_filename

@app.route('/upload', methods=['POST'])
def upload():
    file = request.files.get('file')
    if not file or not file.filename.endswith('.csv'):
        return jsonify(error="CSV file required"), 400
    filename = secure_filename(file.filename)
    file.save(os.path.join(app.config['UPLOAD_FOLDER'], filename))

# 4. Security Headers
@app.after_request
def add_security_headers(response):
    response.headers['X-Content-Type-Options'] = 'nosniff'
    response.headers['X-Frame-Options'] = 'DENY'
    return response

📖 Production Deployment Checklist

Checklist
Pre-Deployment:
  ☐ Set DEBUG = False in production
  ☐ Use environment variables for secrets (SECRET_KEY, DB credentials)
  ☐ Run all tests: pytest tests/
  ☐ Check for SQL injection vulnerabilities (use parameterized queries ✓)
  ☐ Validate all user inputs

Deployment Options:
  1. Gunicorn + Nginx (Linux server)
     $ gunicorn -w 4 -b 0.0.0.0:8000 "app:create_app()"

  2. Docker
     FROM python:3.11-slim
     COPY requirements.txt .
     RUN pip install -r requirements.txt
     COPY . /app
     WORKDIR /app
     CMD ["gunicorn", "-w", "4", "-b", "0.0.0.0:8000", "app:create_app()"]

  3. Cloud (AWS/GCP/Azure)
     - AWS Elastic Beanstalk
     - Google App Engine
     - Azure App Service

Post-Deployment:
  ☐ Set up logging (structured JSON logs)
  ☐ Monitor API response times
  ☐ Set up database backups
  ☐ Configure HTTPS (SSL/TLS certificate)
