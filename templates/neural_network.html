📖 Building a Neural Network (PyTorch Pseudocode)

Python
import torch
import torch.nn as nn

class PriceNet(nn.Module):
    def __init__(self, input_size=5):
        super().__init__()
        self.network = nn.Sequential(
            nn.Linear(input_size, 64),
            nn.ReLU(),
            nn.BatchNorm1d(64),
            nn.Dropout(0.2),
            nn.Linear(64, 32),
            nn.ReLU(),
            nn.Dropout(0.2),
            nn.Linear(32, 1)   # Single output: price
        )

    def forward(self, x):
        return self.network(x)

# Training loop
model = PriceNet()
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
criterion = nn.MSELoss()

for epoch in range(100):
    for batch_X, batch_y in dataloader:
        # Forward pass
        predictions = model(batch_X)
        loss = criterion(predictions, batch_y)

        # Backward pass
        optimizer.zero_grad()
        loss.backward()        # Compute gradients
        optimizer.step()       # Update weights

    if epoch % 10 == 0:
        print(f"Epoch {epoch}: Loss = {loss.item():.4f}")

When to Use Neural Networks vs Traditional ML

Concept
Use Traditional ML (sklearn) when:
  ✅ Structured/tabular data (CSV, SQL)
  ✅ Small-medium dataset (<100K rows)
  ✅ Need interpretability
  ✅ Limited compute resources
  → Our price prediction uses Random Forest

Use Neural Networks when:
  ✅ Unstructured data (images, text, audio)
  ✅ Large dataset (>100K rows)
  ✅ Complex non-linear patterns
  ✅ GPU available
  → Image classification, NLP tasks

📖 CNN for Image Classification

Python
class ImageClassifier(nn.Module):
    """CNN to classify urban scenes."""
    def __init__(self, num_classes=5):
        super().__init__()
        self.features = nn.Sequential(
            # Conv Block 1: detect edges, colors
            nn.Conv2d(3, 32, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2),    # 128x128 → 64x64

            # Conv Block 2: detect shapes, textures
            nn.Conv2d(32, 64, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2),    # 64x64 → 32x32

            # Conv Block 3: detect complex patterns
            nn.Conv2d(64, 128, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2),    # 32x32 → 16x16
        )
        self.classifier = nn.Sequential(
            nn.Flatten(),
            nn.Linear(128 * 16 * 16, 256),
            nn.ReLU(),
            nn.Dropout(0.5),
            nn.Linear(256, num_classes)
        )

    def forward(self, x):
        x = self.features(x)
        return self.classifier(x)

⚠️ Common Mistakes
  • Using deep learning for tabular data: Random Forest often outperforms NNs on structured data
  • Not normalizing inputs: Features on different scales cause training instability
  • Training too long: Monitor validation loss — stop when it starts increasing (early stopping)
