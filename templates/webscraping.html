📖 API Data Extraction

Python
import requests
import pandas as pd

# Example: REST API data extraction
# Note: Replace with your actual API endpoint
url = "https://api.example.com/v1/data"
params = {
    "dataset": "housing_prices",
    "limit": 100
}

response = requests.get(url, params=params, timeout=30)
if response.status_code == 200:
    data = response.json()
    records = data['result']['records']
    df = pd.DataFrame(records)
    print(f"Loaded {len(df)} records")
else:
    print(f"Error: {response.status_code}")

📖 HTML Scraping with BeautifulSoup

Python
from bs4 import BeautifulSoup
import requests

# Example: Scraping public information (educational purpose)
url = "https://en.wikipedia.org/wiki/List_of_metro_systems"

response = requests.get(url, timeout=30)
soup = BeautifulSoup(response.content, 'html.parser')

# Find all tables
tables = soup.find_all('table', class_='wikitable')

# Or use pandas for direct table extraction
dfs = pd.read_html(url)
mrt_table = dfs[0]  # First table on the page
print(mrt_table.head())

# IMPORTANT: Always respect robots.txt and rate limits
import time
time.sleep(1)  # Be polite — wait between requests

📖 Ethical Scraping Guidelines

Important: Always follow these ethical guidelines when scraping data.
  • ✅ Check robots.txt before scraping any website
  • ✅ Respect rate limits — add delays between requests
  • ✅ Use official APIs when available (open data portals, transit data APIs)
  • ✅ Cache responses to avoid repeated requests
  • ✅ Only scrape publicly available information
  • ❌ Don't scrape personal data without consent
  • ❌ Don't overwhelm servers with rapid requests
  • ❌ Don't violate terms of service
