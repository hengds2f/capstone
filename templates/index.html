🗺️ Learning Path

This platform covers the full data science lifecycle, organized as a curriculum:

  1. Data Acquisition: Ingestion, scraping, API calls → Load Singapore open datasets
  2. Storage & Schema: SQL, database design, data warehouse → Star schema with HDB data
  3. Processing: Pandas, NumPy, out-of-core, distributed → Clean and transform data
  4. Analysis: EDA, statistics, visualization → Explore Singapore housing and transport trends
  5. Machine Learning: Supervised, unsupervised, time series → Predict HDB prices, cluster neighborhoods
  6. Deep Learning & AI: Neural networks, CV, NLP → Classify images and text
  7. Engineering: Pipelines, streaming, orchestration → Build production-grade data flows
  8. Deployment: Flask APIs, testing, monitoring → Serve models and dashboards
