📖 NumPy — Numerical Computing Foundation

NumPy provides fast, vectorized operations on arrays. It's the backbone of scientific computing in Python, powering Pandas, scikit-learn, and TensorFlow under the hood.

🎯 Singapore Example: HDB Price Analysis with NumPy

Python
import numpy as np

# HDB resale prices for different towns
prices = np.array([308000, 430000, 560000, 295000, 468000, 545000, 620000, 710000])
areas = np.array([68, 91, 121, 67, 93, 122, 92, 119])

# Vectorized operations (no loops needed!)
price_per_sqm = prices / areas          # Element-wise division
mean_price = np.mean(prices)            # $492,000
std_price = np.std(prices)              # Spread of prices
above_avg = prices[prices > mean_price] # Boolean indexing

# Statistical summary
print(f"Mean: ${mean_price:,.0f}")       # Mean: $492,000
print(f"Std: ${std_price:,.0f}")         # Std: $138,744
print(f"Min PSM: ${np.min(price_per_sqm):,.0f}/sqm")
print(f"Max PSM: ${np.max(price_per_sqm):,.0f}/sqm")

# Matrix operations for feature engineering
features = np.column_stack([areas, price_per_sqm])
correlation = np.corrcoef(areas, prices)  # Correlation matrix
print(f"Area-Price correlation: {correlation[0,1]:.4f}")

📖 Pandas — Data Analysis Powerhouse

Pandas provides DataFrames for structured data manipulation. Think of it as "Excel for programmers" — but much more powerful.

🎯 Loading and Exploring HDB Data

Python
import pandas as pd

# Load CSV data
df = pd.read_csv('data/hdb_resale.csv')

# Quick exploration
print(df.shape)              # (70, 11) — rows, columns
print(df.dtypes)             # Data types for each column
print(df.describe())         # Statistical summary
print(df.isnull().sum())     # Check for missing values
print(df.head())             # First 5 rows

# Data cleaning pipeline
df['resale_price'] = pd.to_numeric(df['resale_price'], errors='coerce')
df['month_dt'] = pd.to_datetime(df['month'], format='%Y-%m')
df['year'] = df['month_dt'].dt.year
df['price_per_sqm'] = (df['resale_price'] / df['floor_area_sqm']).round(2)

# Group-by analysis
town_summary = df.groupby('town').agg(
    avg_price=('resale_price', 'mean'),
    count=('resale_price', 'count'),
    avg_area=('floor_area_sqm', 'mean')
).round(0).sort_values('avg_price', ascending=False)

print(town_summary.head(10))

🔧 Data Transformation Pipeline

Python
# Method chaining for clean transformation pipeline
result = (
    df
    .dropna(subset=['resale_price'])              # Remove null prices
    .assign(
        price_per_sqm=lambda x: x['resale_price'] / x['floor_area_sqm'],
        is_mature=lambda x: x['town'].isin(['BISHAN', 'TOA PAYOH', 'ANG MO KIO',
                                             'QUEENSTOWN', 'BUKIT MERAH']),
        price_band=lambda x: pd.cut(x['resale_price'],
                                     bins=[0, 400000, 600000, 800000, float('inf')],
                                     labels=['Budget', 'Mid', 'Premium', 'Luxury'])
    )
    .query('resale_price > 0')                     # Filter valid prices
    .sort_values('resale_price', ascending=False)   # Sort by price
)

# Pivot table
pivot = df.pivot_table(
    values='resale_price',
    index='town',
    columns='flat_type',
    aggfunc='mean'
).round(0)

print(pivot)
⚠️ Common Mistakes
  • Chained indexing: `df[condition]['col'] = val` causes SettingWithCopyWarning. Use `.loc[]` instead.
  • Ignoring dtypes: Numeric columns loaded as strings — always check with `.dtypes`
  • Memory waste: Use `category` dtype for columns with few unique values (e.g., flat_type)
  • Loop anti-pattern: Using `for` loops on DataFrames instead of vectorized operations
